Classe principal:

Funções/métodos necessários:
- Nenhuma função/método específico é necessário para este exemplo.

Arquivo HTML:
- Nome do arquivo: "table.html"
- Conteúdo do arquivo:

```html
<!DOCTYPE html>
<html>
<head>
	<title>Tabela de Exemplo</title>
	<style>
		table {
			border-collapse: collapse;
			width: 100%;
		}
		th, td {
			text-align: left;
			padding: 8px;
			border-bottom: 1px solid #ddd;
		}
		tr:hover {
			background-color: #f5f5f5;
		}
		th {
			background-color: #4CAF50;
			color: white;
		}
	</style>
</head>
<body>
	<table>
		<thead>
			<tr>
				<th>Nome</th>
				<th>Idade</th>
				<th>Profissão</th>
			</tr>
		</thead>
		<tbody>
			<tr>
				<td>João</td>
				<td>30</td>
				<td>Engenheiro</td>
			</tr>
			<tr>
				<td>Maria</td>
				<td>25</td>
				<td>Advogada</td>
			</tr>
			<tr>
				<td>Carlos</td>
				<td>40</td>
				<td>Médico</td>
			</tr>
		</tbody>
	</table>
</body>
</html>
```

Arquivo CSS:
- Nome do arquivo: "style.css"
- Conteúdo do arquivo:

```css
/* Estilos para a tabela */
table {
	border-collapse: collapse;
	width: 100%;
}
th, td {
	text-align: left;
	padding: 8px;
	border-bottom: 1px solid #ddd;
}
tr:hover {
	background-color: #f5f5f5;
}
th {
	background-color: #4CAF50;
	color: white;
}
```

Certifique-se de salvar o arquivo HTML e o arquivo CSS na mesma pasta. Para visualizar a tabela, basta abrir o arquivo "table.html" em um navegador.
